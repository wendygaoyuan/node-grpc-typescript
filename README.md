# node-grpc-typescript
Node.js gRPC for TypeScript example

# Installation
``` bash
yarn
```

## Build
```bash
# build protos/*.proto
yarn build:proto

# build *.ts
yarn build
```

## Server Start

```sh
# start dev
yarn start

# start prod
yarn prod:server
```

## Client Start

```sh
# start dev
yarn start:client

# start prod
yarn prod:client
```
